void loop (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Arduino main loop. 

When serial data is available:\+ 1) Read three integers using Serial.\+parse\+Int() -\/\texorpdfstring{$>$}{>} r, g, b 2) Remove leftover newline/\+carriage return characters from the buffer 3) Print parsed values for debugging 4) Update the RGB LED using set\+RGB(r, g, b)

\begin{DoxyNote}{Note}
Serial.\+parse\+Int() skips non-\/numeric characters (like commas), so input formats like {\ttfamily 200,100,40} or {\ttfamily 200 100 40} both work. 
\end{DoxyNote}
